# AT&T Data Science Club (Bothell)
Welcome to the official repository for Bothell's Data Science Club! This repo will contain the walkthrough worksheets for each meeting, plus answer keys.

In order to run the worksheets for each of the meetings, follow these steps:
1. First, go to Binder (https://mybinder.org/) and insert the link to this repo.
2. Use the requirements.txt, loading in any necessary libraries to run the notebook.
